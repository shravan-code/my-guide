🎯 What is Apache Spark?

Apache Spark is an open-source, distributed computing system for fast, in-memory analytics. It supports batch, streaming, machine learning, and interactive queries with a unified API.

Comparison: Spark vs Hadoop MapReduce
- Speed: Spark in-memory vs disk-based
- APIs: High-level DataFrame API vs low-level MapReduce
- SQL: Spark SQL vs Hive on Hadoop
- Streaming: Structured Streaming in Spark
📜 Evolution of Apache Spark
From research project to industry standard

🔑 SparkContext & SparkSession

SparkContext is the primary entry point to Spark functionality, representing the connection to a Spark cluster. SparkSession is the unified entry point that encompasses SparkContext and SQL features.

# SparkSession (preferred)
from pyspark.sql import SparkSession

spark = SparkSession.builder \
  .appName("MyApp") \
  .master("local[*]") \
  .getOrCreate()

sc = spark.sparkContext

# SparkContext (older style)
from pyspark import SparkContext
sc = SparkContext("local[*]", "MyApp")

Interview Questions & Examples

🔰 Basic Level Questions
1 What is PySpark and how does it differ from Apache Spark? Basic
💡 Answer

PySpark is the Python API for Apache Spark. Apache Spark is the core engine; PySpark provides a Pythonic interface to Spark's distributed processing framework. Differences are mainly language bindings and serialization between Python and the JVM-based Spark engine.

2 What is SparkContext and SparkSession? Basic
💡 Answer

SparkContext is the entry point to Spark functionality (older API). SparkSession is the unified entry point since Spark 2.x, combining SQLContext and SparkContext.

🚀 Advanced Level Questions
16 What is the Catalyst Optimizer and how does it work? Advanced
💡 Answer

The Catalyst Optimizer is Spark’s query optimizer. It analyzes and rewrites logical plans into efficient physical plans, applying rules for predicate pushdown, constant folding, and more. Phases include Analysis, Logical Optimization, Physical Planning, and Cost-based optimization.
