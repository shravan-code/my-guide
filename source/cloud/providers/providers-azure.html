Azure Overview Basic

Microsoft Azure is the world's leading enterprise cloud platform with 200+ services. It offers strong integration with Microsoft ecosystem and enterprise features.

Key Facts

  • Launch: 2010
  • Market Share: ~23% (second largest)
  • Regions: 60+ worldwide
  • Services: 200+
  • Free Tier: 12 months + always free

Key Benefits

  • Microsoft ecosystem integration
  • Enterprise-grade security
  • Hybrid cloud (Azure Arc)
  • Strong compliance certifications
  • Open source friendly

Machine Learning & AI Intermediate

Azure provides end-to-end ML through Azure ML Studio and pre-built AI services.

# Azure Machine Learning # SDK v2 from azure.ai.ml import MLClient from azure.ai.ml import command from azure.identity import DefaultAzureCredential # Connect to workspace credential = DefaultAzureCredential() ml_client = MLClient(credential, subscription_id, resource_group, workspace_name) # Create compute compute = ml_client.compute.get_or_create( name="cpu-cluster", type="amlcompute", size="Standard_DS3_v2", min_instances=0, max_instances=4 ) # Submit training job job = command( code="./src", command="python train.py", environment="azureml:sklearn-1.0@latest", compute="cpu-cluster" ) returned_job = ml_client.jobs.create_or_update(job) # Deploy endpoint # Online (real-time) or Batch endpoints # Cognitive Services / Azure AI Services # Pre-built AI models Text Analytics # Sentiment, NER, key phrases Vision # Image analysis, OCR Speech # STT, TTS, transcription Translator # Machine translation Document Intell. # Form/document extraction Custom Vision # Custom image classification Decision # Content moderation # Azure OpenAI Service # GPT-4, GPT-3.5, Embeddings, DALL-E az cognitiveservices account create \ --name myopenai \ --resource-group myRG \ --kind OpenAI \ --sku S0 \ --location eastus # Azure AI Search # Enterprise search with AI enrichment # Vector search, semantic search az search service create \ --name mysearch \ --resource-group myRG \ --sku Standard

Azure CLI Basic

Azure CLI (az) is the cross-platform command-line tool for managing Azure resources.

# Installation # Windows winget install Microsoft.AzureCLI # macOS brew install azure-cli # Linux curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash # Login az login # Browser-based az login --service-principal # Service principal az login --identity # Managed Identity # Set subscription az account set --subscription "My Subscription" # Common CLI Commands # Resource Groups az group create --name myRG --location eastus az group list --output table az group delete --name myRG --yes # VMs az vm list --output table az vm start --resource-group myRG --name myVM az vm stop --resource-group myRG --name myVM az vm deallocate --resource-group myRG --name myVM # Storage az storage account list --output table az storage blob list --account-name mystorage --container-name mycontainer # Functions az functionapp list --output table az functionapp start --resource-group myRG --name myapp # AKS az aks list --output table az aks get-credentials --resource-group myRG --name myAKS # Useful commands az version # CLI version az account show # Current account az account list-locations --output table # Available regions # Output formats --output json # JSON (default) --output table # Table format --output yaml # YAML format --output tsv # Tab-separated # Query with JMESPath az vm list --query "[].{Name:name, Location:location, Size:hardwareProfile.vmSize}" --output table

Bicep & Terraform Intermediate

Define Azure infrastructure using Bicep (Azure DSL) or Terraform (multi-cloud IaC).

# Bicep - Azure's DSL for infrastructure # main.bicep param location string = resourceGroup().location param storageAccountName string = 'st${uniqueString(resourceGroup().id)}' resource storageAccount 'Microsoft.Storage/storageAccounts@2023-01-01' = { name: storageAccountName location: location kind: 'StorageV2' sku: { name: 'Standard_LRS' } } resource functionApp 'Microsoft.Web/sites@2023-01-01' = { name: 'func-${uniqueString(resourceGroup().id)}' location: location kind: 'functionapp' properties: { siteConfig: { appSettings: [ { name: 'AzureWebJobsStorage' value: storageAccount.properties.primaryEndpoints.blob } ] } } } # Deploy Bicep az deployment group create \ --resource-group myRG \ --template-file main.bicep # ARM Templates (JSON) - Generated from Bicep # Bicep is transpiled to ARM JSON # Terraform - Multi-cloud IaC # main.tf provider "azurerm" { features {} } resource "azurerm_resource_group" "example" { name = "myResourceGroup" location = "East US" } resource "azurerm_storage_account" "example" { name = "mystorageaccount" resource_group_name = azurerm_resource_group.example.name location = azurerm_resource_group.example.location account_tier = "Standard" account_replication_type = "LRS" } # Terraform commands terraform init terraform plan terraform apply terraform destroy

Monitoring & Observability Intermediate

Azure Monitor provides comprehensive monitoring with metrics, logs, and alerts.

# Azure Monitor - Unified monitoring # Metrics Explorer # Built-in metrics for all Azure resources # VM CPU, Storage transactions, DB DTUs # Log Analytics - Query logs with KQL # Kusto Query Language Heartbeat | where TimeGenerated > ago(1h) | summarize count() by Computer, _ResourceId | order by count_ desc # Application Insights - APM # Application performance monitoring from azure.monitor.opentelemetry import AzureMonitorOpentelemetry # Python FastAPI example from opentelemetry.instrumentation.fastapi import FastAPIInstrumentor FastAPIInstrumentor.instrument_app(app) # Alerts az monitor metrics alert create \ --name "High CPU" \ --resource-group myRG \ --condition "avg Percentage CPU > 80" \ --scopes /subscriptions/.../vm/myVM # Azure Log Analytics workspace az monitor log-analytics workspace create \ --resource-group myRG \ --workspace-name myWorkspace # KQL Queries # Application Performance requests | where success == true | summarize avgDuration = avg(duration) by bin(timestamp, 1h) | render timechart # Infrastructure Heartbeat | summarize LastHeartbeat = max(TimeGenerated) by Computer | where LastHeartbeat < ago(5m) # Other Monitoring Services Application Insights # APM for web apps Network Watcher # Network monitoring Service Health # Azure service status Advisor # Best practice recommendations Cost Management # Spending analysis
