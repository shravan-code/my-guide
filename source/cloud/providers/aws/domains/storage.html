S3 & Data Lake Services

Deep dive into S3, Lake Formation, and data lake architectures.

S3 - Simple Storage Service

What It Does

Object storage service with 99.999999999% (11 9's) durability. Stores any amount of data as objects in buckets. Supports versioning, encryption, lifecycle policies, and cross-region replication.

Why Use It

Foundation of modern data architecture. Unlimited scalability, pay-per-use pricing, and integration with every AWS analytics service. Supports multiple storage classes for cost optimization.

Storage Classes

  • S3 Standard: Frequently accessed, high durability
  • S3 Intelligent-Tiering: Auto-move based on access
  • S3 Standard-IA: Infrequent access, lower cost
  • S3 Glacier: Archival, retrieval in minutes to hours
  • S3 Glacier Deep Archive: Long-term, 7-10 year retention

Use Cases

  • Data Lake raw storage
  • ETL staging and output
  • Machine learning training data
  • Backup and disaster recovery
  • Static website hosting

Lake Formation - Data Lake Governance

What It Does

Centralized data lake governance built on Glue Catalog. Provides fine-grained access control, row-level security, data masking, and automated data ingestion.

Why Use It

Simplifies secure data lake setup from weeks to days. Enforces consistent security policies across all data sources. Integrates with IAM, SSO, and active directories.

Key Features

  • Fine-grained access: Table and column-level permissions
  • Row-level security: Data cells filtering
  • Data masking: Mask sensitive fields
  • Blueprints: Automated ingestion templates
  • Cross-account access: Share data securely

EFS - Elastic File System

What It Does

Managed NFS file storage for EC2 instances. Scalable on-demand without provisioning. Supports thousands of concurrent connections.

Why Use It

Shared storage for Spark clusters, ML training, and big data workloads. Unlike S3, supports POSIX file system semantics.

Use Cases

  • Spark HDFS replacement for EMR
  • Shared ML training data
  • Container storage (ECS/EKS)
  • CI/CD build artifacts
