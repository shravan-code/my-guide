Data Warehouse & Analytics

Analytics & Business Intelligence

Redshift, Athena, and QuickSight for data warehousing and visualization.

Amazon Redshift

What It Does

Petabyte-scale data warehouse using columnar storage and MPP (Massively Parallel Processing). Stores data in dense compute nodes with automatic compression and zone maps.

Why Use It

Best-in-class SQL performance on structured data. RA3 allows scaling compute and storage independently. Integrates with BI tools and supports ML.

Distribution Keys

// Distribution Key (DISTKEY) - Where data lives KEY: Co-locate JOIN keys (fact-dimension joins) ALL: Copy small tables to all nodes (< 5MB) EVEN: Round-robin (default, no clear pattern) // Example: sales fact table CREATE TABLE sales (...) DISTKEY(product_id); // Example: small dimension table CREATE TABLE product_dim (...) DISTSTYLE ALL;

Sort Keys

// Sort Key - Data ordering for zone maps COMPOUND: First column used for pruning (most common) INTERLEAVED: Equal weight to all columns (rare cases) // Example CREATE TABLE events (...) COMPOUND SORTKEY(event_date, event_type); // Best for: WHERE event_date > '2024-01-01'

Amazon Athena

What It Does

Serverless SQL query service for S3 data. Presto-based, pay per query. No infrastructure to manage, instant scaling.

Why Use It

Query data in S3 without loading. Perfect for ad-hoc analysis, data exploration, and data lake queries.

Use Cases

  • Ad-hoc queries on S3 data
  • Log analysis (CloudTrail, VPC Flow Logs)
  • Data lake queries
  • Quick exploration without data loading

Amazon QuickSight

What It Does

Cloud-native BI service for interactive dashboards. Supports ML insights, natural language queries, and embedded analytics.

Why Use It

Fully managed, scales automatically. Connects to Redshift, S3, Athena, RDS. SPICE in-memory engine for fast performance.

Features

  • ML Insights: Anomaly detection, forecasting
  • Q: Natural language queries
  • SPICE: Fast in-memory analytics
  • Embedded: Embed dashboards in apps
  • Paginated: Pixel-perfect reports
