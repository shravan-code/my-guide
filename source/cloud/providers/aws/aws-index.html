9-Layer Architecture

Every data pipeline follows these layers:

1 Data Source RDS, S3, APIs, IoT
2 Ingestion DMS, Kinesis, Glue, DataSync
3 Processing Glue, EMR, Lambda, Step Functions
4 Storage S3, Lake Formation, DynamoDB
5 Orchestration MWAA, Step Functions, EventBridge
6 Query & Analytics Athena, Redshift, QuickSight
7 Monitoring CloudWatch, CloudTrail, X-Ray
8 Security IAM, KMS, Secrets Manager, VPC
9 Catalog & Metadata Glue Catalog, Lake Formation
