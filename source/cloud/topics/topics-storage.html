Cloud Storage

Object, block, file, and data lake storage solutions across AWS, Azure, and GCP.

Object Storage Basic

Object storage is the foundation of modern cloud data architectures. It stores unstructured data as objects in buckets with unlimited scalability.

cloud AWS S3

99.999999999% durability. Multiple storage classes (Standard, IA, Glacier). Lifecycle policies, versioning, cross-region replication.

cloud Azure Blob

Hot/Cool/Archive tiers. Data Lake Gen2 for analytics. Supports hierarchical namespace for filesystem-like access.

cloud GCP Cloud Storage

Standard/Nearline/Coldline/Archive classes. Dual-region and multi-region options. Immutable retention policies.

# AWS S3 CLI Examples # Create bucket aws s3 mb s3://my-data-lake --region us-east-1 # Upload file aws s3 cp data.csv s3://my-bucket/raw/ # Set lifecycle policy aws s3 put-bucket-lifecycle-configuration \ --bucket my-bucket \ --lifecycle-configuration file://lifecycle.json

Block Storage Basic

Block storage provides high-performance, low-latency storage for VMs and databases. Attached as virtual disks.

AWS EBS

  • gp3/gp2 - General Purpose SSD
  • io2 - Provisioned IOPS SSD
  • st1 - Throughput Optimized HDD
  • sc1 - Cold HDD

Azure Managed Disks

  • Premium SSD
  • Standard SSD
  • Standard HDD
  • Ultra Disks

File Storage Basic

File storage provides shared file systems accessible via NFS or SMB protocols. Ideal for workloads requiring filesystem semantics.

folder AWS EFS

Elastic file system. Scales automatically. Supports thousands of concurrent connections. Pay-per-use pricing.

folder Azure Files

SMB 3.0 and REST API. Supports Azure File Sync. Hot/Cool tiers available.

folder GCP Filestore

High-performance file storage. SSD and HDD options. NFS v3 protocol support.

Data Lake Storage Intermediate

Data lakes store raw data in native format at any scale. Enable analytics, ML, and real-time processing.

database AWS Lake Formation

Build secure data lakes in days. Fine-grained access control. Blueprints for automated ingestion.

database Azure Data Lake

ADLS Gen2 with hierarchical namespace. Azure Synapse integration. Enterprise-grade security.

database GCP Cloud Storage + Dataplex

Data lake powered by GCS. Dataplex provides metadata management and data fabric across domains.
