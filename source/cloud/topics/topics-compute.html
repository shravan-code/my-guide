Cloud Compute

Virtual machines, containers, Kubernetes, and serverless computing across cloud providers.

Virtual Machines Basic

Virtual machines provide configurable compute capacity. Each cloud offers managed VM services with different instance types.

dns AWS EC2

Elastic Compute Cloud. 750+ instance types. Auto Scaling, Load Balancing, Spot instances.

dns Azure VMs

Virtual Machines. Azure VM Scale Sets. Confidential computing support. Spot VMs.

dns GCP Compute Engine

Preemptible VMs. Sole-tenant nodes. Live migration. Custom machine types.

Kubernetes Intermediate

Managed Kubernetes services abstract cluster management while providing full Kubernetes API access.

hub AWS EKS

Elastic Kubernetes Service. Fargate integration. Karpenter for auto-scaling. Cluster Explorer.

hub Azure AKS

Azure Kubernetes Service. Azure AD integration. Azure Policy. GitOps with Flux.

hub GCP GKE

Google Kubernetes Engine. Autopilot mode. Cloud Run integration. Config Sync.

Serverless Intermediate

Serverless computing runs code without provisioning servers. Pay only for compute time consumed.

bolt AWS Lambda

Run code for virtually any application. 15 min timeout. 10GB memory. 1000 concurrent executions.

bolt Azure Functions

Event-driven compute. Durable Functions for stateful workflows. Consumption/Premium plans.

bolt GCP Cloud Functions

HTTP and event-triggered functions. 2nd gen runtime. Up to 60 min timeout. Auto-scaling.
