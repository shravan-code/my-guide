What is Python? Getting Started

Python is a high-level, interpreted programming language known for its clean syntax and readability. Created by Guido van Rossum in 1991, it emphasizes code simplicity and supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Memory Basics Memory

Python handles memory automatically, but understanding how it works helps write efficient code.

# Everything is an object in Python # Each object has: type, reference count, value # Memory addresses (id) a = [1, 2, 3] b = a print(id(a) == id(b)) # True - same object # Shallow vs Deep copy import copy original = [[1, 2], [3, 4]] shallow = copy.copy(original) # New list, same nested objects deep = copy.deepcopy(original) # Completely independent copy # Check memory usage import sys print(sys.getsizeof("hello")) # Bytes used by string object

Garbage Collection Memory

Python automatically reclaims memory by cleaning up objects no longer in use.

Reference Counting

Each object tracks how many references point to it. When count reaches zero, memory is freed immediately.

Generational GC

Objects are grouped by age. Young objects are collected frequently; old objects less often.

# Circular references can prevent immediate cleanup a = {} b = {} a['next'] = b b['prev'] = a del a, b # GC must detect and clean circular refs # Manual garbage collection import gc gc.collect() # Force garbage collection # Check reference count print(sys.getrefcount(a) - 1) # Subtract reference from getrefcount itself

Memory Optimization Memory

Techniques to reduce memory usage and improve performance.

# Use generators for large sequences (lazy evaluation) def read_large_file(path): with open(path) as f: for line in f: yield line # One line at a time, not all in memory # List vs Generator memory import sys numbers = [x ** 2 for x in range(1000000)] gen = (x ** 2 for x in range(1000000)) print(sys.getsizeof(numbers)) # ~8 MB print(sys.getsizeof(gen)) # ~200 bytes! # Use __slots__ to reduce class memory class Point: __slots__ = ['x', 'y'] # Prevents __dict__ creation def __init__(self, x, y): self.x = x self.y = y # Weak references - don't prevent GC import weakref data = [1, 2, 3] weak = weakref.ref(data) print(weak()) # [1, 2, 3] del data print(weak()) # None - object was garbage collected

Big O Notation Performance

Describes how algorithm performance scales with input size. Critical for data engineering.

  • O(1) - Constant: Array index access, hash lookup
  • O(log n) - Logarithmic: Binary search in sorted data
  • O(n) - Linear: Simple loop, list iteration
  • O(n log n) - Linearithmic: Efficient sorting (merge, quick)
  • O(n²) - Quadratic: Nested loops, naive sorting
  • O(2^n) - Exponential: Recursive fibonacci without memoization
# O(1) - Hash lookup from collections import defaultdict cache = defaultdict(lambda: 0) cache["key"] = 42 print(cache["key"]) # Instant lookup regardless of size # O(n) - Linear search def find_max(items): max_val = items[0] for item in items: if item > max_val: max_val = item return max_val # O(n log n) - Sorting before searching import bisect sorted_list = [1, 3, 5, 7, 9] idx = bisect.bisect_left(sorted_list, 5) print(idx) # 2 - position of 5

File Formats Storage

Choosing the right format impacts performance and storage efficiency.

CSV / TSV

Human-readable, universal compatibility. Slow for large files, no schema enforcement.

JSON

Hierarchical data, web APIs. Verbose syntax, larger file sizes.

Parquet

Columnar storage, compressed. Excellent for analytics, requires specialized tools.

ORC

Optimized row columnar. Similar to Parquet, Hive-optimized.

# Reading different formats with Pandas import pandas as pd # CSV - universal but slow df = pd.read_csv("data.csv") # JSON - nested data df = pd.read_json("data.json") # Parquet - best for analytics df = pd.read_parquet("data.parquet") # Excel files df = pd.read_excel("data.xlsx", sheet_name="Sheet1") # Writing with compression df.to_parquet("data.parquet.gz", compression="gzip")

Data Serialization Storage

Convert Python objects to bytes for storage or transmission.

# Pickle - Python object serialization import pickle data = {"name": "Alice", "scores": [90, 85, 92]} with open("data.pkl", "wb") as f: pickle.dump(data, f) with open("data.pkl", "rb") as f: loaded = pickle.load(f) # JSON - cross-language compatible import json json_str = json.dumps(data) parsed = json.loads(json_str) # MessagePack - compact binary format # pip install msgpack import msgpack packed = msgpack.packb(data) unpacked = msgpack.unpackb(packed, raw=False) # Arrow - columnar format for analytics import pyarrow as pa import pyarrow.parquet as pq table = pa.Table.from_pandas(df) pq.write_table(table, "data.parquet")
