Python Fundamentals

What is Python?

Python is a high-level, interpreted programming language known for its clean syntax and readability. Created by Guido van Rossum in 1991, it emphasizes code simplicity and supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Why Python for Data Engineering?

Python has become the de facto language for data engineering due to its rich ecosystem of libraries and frameworks designed for data processing, analysis, and machine learning.

Data Processing

Pandas, NumPy, Dask for handling large datasets efficiently.

Big Data

ETL Pipelines

Cloud Integration

Native support for AWS, Azure, GCP services and APIs.

Python uses indentation to define code blocks, making it visually clean and easy to read.

Data Types in Python

Python has several built-in data types that form the foundation of data manipulation.

  • Text Type: str - "Hello World"
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict - {"name": "Alice", "age": 25}
  • Set Types: set, frozenset
  • Boolean Type: bool - True / False
  • Binary Types: bytes, bytearray, memoryview

Collections: Lists and Dictionaries

Lists and dictionaries are the most commonly used data structures in Python for data engineering.

# Lists - ordered, mutable collections numbers = [1, 2, 3, 4, 5] numbers.append(6) print(numbers) # [1, 2, 3, 4, 5, 6] # Dictionaries - key-value pairs person = { "name": "Alice", "age": 25, "city": "New York" } print(person["name"]) # Alice

Control Flow: Conditionals and Loops

# If-else statements temperature = 30 if temperature > 35: print("Hot day") elif temperature > 20: print("Nice weather") else: print("Cold day") # For loop for i in range(5): print(i) # 0, 1, 2, 3, 4 # List comprehension squares = [x**2 for x in range(10)]

Functions and Modules

Functions help organize code into reusable blocks. Modules are files containing Python code that can be imported.

# Defining a function def calculate_average(numbers): """Calculate the average of a list of numbers.""" return sum(numbers) / len(numbers) # Using the function avg = calculate_average([10, 20, 30]) print(avg) # 20.0 # Importing modules import pandas as pd from datetime import datetime

Working with Files

Python makes it easy to read from and write to files, essential for ETL pipelines.

# Reading a CSV file with Pandas import pandas as pd df = pd.read_csv("data.csv") print(df.head()) # Writing to a file df.to_csv("output.csv", index=False) # Reading with context manager with open("file.txt", "r") as f: content = f.read()
