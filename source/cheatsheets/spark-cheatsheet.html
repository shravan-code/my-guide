Spark Cheatsheet

Quick reference for PySpark DataFrame operations.

Data Structures Basic

DataFrame

DataFrame

Distributed collection of data organized into named columns.

from pyspark.sql import SparkSession spark = SparkSession.builder.getOrCreate() # Create DataFrame data = [("Alice", 25), ("Bob", 30)] df = spark.createDataFrame(data, ["Name", "Age"])
RDD

RDD (Resilient Distributed Dataset)

Low‑level distributed collection of objects (legacy API).

from pyspark import SparkContext sc = SparkContext.getOrCreate() rdd = sc.parallelize([1, 2, 3, 4, 5]) squared = rdd.map(lambda x: x**2)

Session & Configuration Basic

Session

Create Spark Session

Initialize Spark with configuration options.

from pyspark.sql import SparkSession spark = SparkSession.builder \ .appName("MyApp") \ .config("spark.sql.shuffle.partitions", "200") \ .getOrCreate()
Config

Common Configurations

Useful Spark configurations.

spark.conf.set("spark.executor.memory", "8g") spark.conf.set("spark.sql.adaptive.enabled", "true")

Reading Data Medium

Parquet

Read Parquet

Read Parquet files with options.

df = spark.read.parquet("path/")
CSV

Read CSV

Read CSV files with options.

df = spark.read.option("header", "true").csv("path/")
JDBC

Read from Database

Read from relational databases.

df = spark.read.format("jdbc").option("url", url).load()
JSON

Read JSON

Read JSON files.

df = spark.read.json("path/")

Transformations Medium

Select

Select & WithColumn

Select columns and add new ones.

df.select("name") df.withColumn("new", F.lit(1))
Filter

Filter & Where

Filter rows based on conditions.

df.filter(F.col("age") > 18)
GroupBy

Aggregation

Group and aggregate data.

df.groupBy("cat").agg(F.sum("amt"))
Distinct

Distinct & DropDuplicates

Remove duplicate rows.

df.dropDuplicates(["id"])

Joins Advanced

Join Types

Join DataFrames

Join DataFrames with different types.

df1.join(df2, "id", "left")
Conditions

Complex Join Conditions

Joins with multiple conditions.

df1.join(df2, (df1.id == df2.uid), "inner")
Broadcast

Broadcast Join

Optimize joins with small tables.

from pyspark.sql.functions import broadcast df.join(broadcast(small_df), "id")

Write Modes

UDFs & Functions Advanced

UDF

User Defined Functions

Create custom functions for Spark.

from pyspark.sql.functions import udf from pyspark.sql.types import StringType, IntegerType # Define UDF @udf(StringType()) def extract_domain(email): if email and "@" in email: return email.split("@")[1] return None # Register UDF spark.udf.register("extract_domain", lambda x: x.split("@")[1] if x else None, StringType()) # Use UDF df.withColumn("domain", extract_domain(F.col("email")))
Pandas UDF

Pandas UDF (Vectorized)

Fast UDFs using pandas/vectorized operations.

from pyspark.sql.functions import pandas_udf from pyspark.sql.types import DoubleType import pandas as pd @pandas_udf(DoubleType()) def pandas_mean(series: pd.Series) -> float: return series.mean() # Apply df.groupBy("category") \ .agg(pandas_mean(F.col("amount"))) # Series to Series @pandas_udf(StringType()) def uppercase_udf(s: pd.Series) -> pd.Series: return s.str.upper()
Window

Window Functions

Analytics with window specifications.

from pyspark.sql.window import Window window = Window.partitionBy("category") \ .orderBy(F.col("amount").desc()) # Rank and row number df.withColumn("rank", F.rank().over(window)) df.withColumn("row_num", F.row_number().over(window)) # Lag and lead df.withColumn("prev_amount", F.lag("amount", 1).over(window)) df.withColumn("next_amount", F.lead("amount", 1).over(window)) # Running total window_unbounded = Window.partitionBy("category") \ .orderBy("date") \ .rowsBetween(Window.unboundedPreceding, 0) df.withColumn("running_total", F.sum("amount").over(window_unbounded))
Built-in

Built-in Functions

Common built-in functions.

# String functions F.upper("name") F.substring("name", 1, 3) F.regexp_extract("text", r"(\d+)", 1) # Date functions F.current_date() F.date_format("date_col", "yyyy-MM-dd") F.datediff("end_date", "start_date") # Conditional F.when(F.col("x") > 0, "positive") \ .when(F.col("x") < 0, "negative") \ .otherwise("zero")

Performance Advanced

Repartition

Repartition & Coalesce

Control partitioning for performance.

# Repartition (shuffle) df.repartition(200) df.repartition("category") # Coalesce (no shuffle) df.coalesce(10) # Cache df.cache()
Caching

Cache & Persist

Cache DataFrames for reuse.

from pyspark import StorageLevel # Cache (in memory) df.cache() # Persist with level df.persist(StorageLevel.MEMORY_AND_DISK) # Unpersist df.unpersist() # Cache table df.createOrReplaceTempView("my_table") spark.catalog.cacheTable("my_table")
Explain

Query Plan

Analyze query execution plans.

# Show plan df.explain("extended") # Print schema df.printSchema() # Stats df.describe().show()
AQE

Adaptive Query Execution

Enable AQE for automatic optimization.

# Enable AQE spark.conf.set("spark.sql.adaptive.enabled", "true") # Coalesce partitions spark.conf.set("spark.sql.adaptive.coalescePartitions.enabled", "true") # Skew join optimization spark.conf.set("spark.sql.adaptive.skewJoin.enabled", "true")

Useful Modules Medium

pyspark.sql

Spark SQL

Structured data processing with DataFrames and SQL.

from pyspark.sql import SparkSession from pyspark.sql.functions import col, avg spark = SparkSession.builder.getOrCreate() # Read JSON df = spark.read.json("data.json") # Query with SQL df.createOrReplaceTempView("people") result = spark.sql("SELECT * FROM people WHERE age > 30")
pyspark.ml

Machine Learning

ML pipelines, feature extraction, and algorithms.

from pyspark.ml import Pipeline from pyspark.ml.feature import VectorAssembler from pyspark.ml.classification import LogisticRegression # Assemble features assembler = VectorAssembler(inputCols=["f1", "f2"], outputCol="features") # Logistic regression lr = LogisticRegression(featuresCol="features", labelCol="label") # Pipeline pipeline = Pipeline(stages=[assembler, lr]) model = pipeline.fit(train_df)
pyspark.streaming

Structured Streaming

Real‑time data processing with continuous queries.

from pyspark.sql.functions import window # Read stream from Kafka df = spark.readStream.format("kafka") \ .option("kafka.bootstrap.servers", "localhost:9092") \ .option("subscribe", "topic").load() # Windowed aggregation windowed = df.groupBy(window(df.timestamp, "10 mins")).count() query = windowed.writeStream.outputMode("append").start()

Tips & Tricks

Tip

Cache intermediate DataFrames

Reuse expensive computations with caching.

# Cache after expensive transformation df = df.filter(...).select(...).cache() # Or persist with storage level from pyspark import StorageLevel df.persist(StorageLevel.MEMORY_AND_DISK)
Trick

Use broadcast joins for small tables

Speed up joins by broadcasting smaller DataFrame.

from pyspark.sql.functions import broadcast # Join with broadcast hint result = large_df.join( broadcast(small_df), on="id", how="inner" )
Tip

Avoid UDFs when possible

Prefer built‑in functions; UDFs have serialization overhead.

# Fast built-in from pyspark.sql.functions import upper df.withColumn("upper", upper(df.name))

Best Practices Medium

Partitioning

Use partition pruning

Filter on partition columns to reduce scanned data.

df = spark.read.parquet('/data/events') \ .filter('event_date = "2024-05-01"')
Shuffle

Reduce shuffle

Aggressively use coalesce/repartition only when needed.

# Repartition only if needed df = df.repartition(200)
Logging

Check logs and history server

Use Spark UI and logs to diagnose stage bottlenecks.

# Spark UI: http://localhost:4040 # History Server for finished apps
