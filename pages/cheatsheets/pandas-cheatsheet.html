Tips & Tricks

Tip

Use vectorized operations

Avoid loops; leverage pandas built‑in vectorization for speed.

# Slow for i in range(len(df)): df.at[i, 'new'] = df.at[i, 'value'] * 2 # Fast df['new'] = df['value'] * 2
Trick

Chaining method calls

Use `.pipe()` to chain custom functions.

def add_column(df, col_name, value): df[col_name] = value return df result = (df .pipe(add_column, 'flag', True) .query('flag == True') .drop('flag', axis=1))
Tip

Use `inplace=True` sparingly

Prefer assignment over inplace operations for clarity.

# Avoid df.dropna(inplace=True) # Prefer df = df.dropna()
Trick

Method chaining with `assign`

Create multiple columns in one chain.

df = (df .assign( log_price=lambda x: np.log(x['price']), price_sq=lambda x: x['price'] ** 2, high_price=lambda x: x['price'] > x['price'].median() ))

Best Practices Medium

Memory

Use categorical types

Convert low-cardinality columns to category for memory savings.

df['category'] = df['category'].astype('category') print(df.memory_usage(deep=True))
Lazy

Chain transformations

Use chained operations with assign/apply to avoid intermediate copies.

df = (df .assign(total=lambda x: x['price'] * x['quantity']) .query('total > 100') .dropna(subset=['customer']))
Integrity

Use `.copy()` carefully

Avoid SettingWithCopy warnings and explicit copy for clarity.

subset = df.loc[df['active'], ['name', 'email']].copy() subset['name'] = subset['name'].str.strip()
