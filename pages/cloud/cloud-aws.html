AWS Services

AWS Overview Basic

Amazon Web Services (AWS) is the world's leading cloud platform with 200+ services. It offers compute, storage, databases, analytics, ML, and more.

Key Facts

  • Launch: 2006 (oldest major provider)
  • Market Share: ~32% (leading)
  • Regions: 30+ worldwide
  • Services: 200+
  • Free Tier: 12 months + always free

Key Benefits

  • Broadest service portfolio
  • Largest partner ecosystem
  • Most mature platform
  • Strong compliance certifications
  • Active open-source community

AWS SDK (Boto3) Basic

Boto3 is the Python SDK for AWS. It provides Pythonic interfaces to all AWS services.

# Installation pip install boto3 botocore # Configuration # ~/.aws/credentials [default] aws_access_key_id = AKIAIOSFODNN7EXAMPLE aws_secret_access_key = wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY # ~/.aws/config [default] region = us-east-1 output = json # Clients vs Resources import boto3 # Low-level Client (raw API calls) client = boto3.client('s3') response = client.list_objects_v2(Bucket='my-bucket') # High-level Resource (Pythonic) s3 = boto3.resource('s3') bucket = s3.Bucket('my-bucket') for obj in bucket.objects.all(): print(obj.key) # Session for multiple profiles session = boto3.Session(profile_name='dev') s3_client = session.client('s3') # Assume Role (cross-account) sts_client = boto3.client('sts') assumed = sts_client.assume_role( RoleArn='arn:aws:iam::123456789:role/MyRole', RoleSessionName='my-session' ) credentials = assumed['Credentials']
# Common Boto3 Examples # S3 Operations import boto3 s3 = boto3.client('s3') # Upload file s3.upload_file('local.txt', 'my-bucket', 'remote.txt') # Download file s3.download_file('my-bucket', 'remote.txt', 'local.txt') # List objects response = s3.list_objects_v2(Bucket='my-bucket', Prefix='data/') for obj in response.get('Contents', []): print(obj['Key'], obj['Size']) # DynamoDB Operations dynamodb = boto3.resource('dynamodb') table = dynamodb.Table('users') # Put item table.put_item(Item={ 'userId': '123', 'name': 'Alice', 'email': 'alice@example.com' }) # Get item response = table.get_item(Key={'userId': '123'}) item = response['Item'] # Query with GSI response = table.query( IndexName='email-index', KeyConditionExpression=Key('email').eq('alice@example.com') ) # Lambda invoke lambda_client = boto3.client('lambda') response = lambda_client.invoke( FunctionName='my-function', Payload=json.dumps({'key': 'value'}) ) result = json.loads(response['Payload'].read())
