GCP Overview

Google Cloud Platform (GCP) offers cloud computing services built on Google's infrastructure. GCP excels in data analytics, machine learning, and open-source technologies like Kubernetes (created at Google).

Key Facts

  • Launch: 2008
  • Market Share: ~11% (third largest)
  • Regions: 40+ worldwide
  • Services: 100+
  • Free Tier: $300 credit (90 days) + always free

Key Strengths

  • Data analytics and BigQuery leadership
  • ML/AI and TPUs
  • Kubernetes (K8s originated here)
  • Open source friendly
  • Sustainable (carbon-neutral since 2007)

Google Cloud Python SDK

The google-cloud-* Python packages provide idiomatic interfaces to all GCP services.

# Installation pip install google-cloud-storage pip install google-cloud-bigquery pip install google-cloud-firestore pip install google-cloud-pubsub pip install google-cloud-aiplatform # Authentication # 1. Application Default Credentials (ADC) # Set GOOGLE_APPLICATION_CREDENTIALS env var import os os.environ["GOOGLE_APPLICATION_CREDENTIALS"] = "path/to/key.json" # 2. Use ADC in code from google.cloud import storage client = storage.Client() # Uses ADC automatically # Cloud Storage operations bucket = client.get_bucket("my-bucket") # Upload blob blob = bucket.blob("data/file.txt") blob.upload_from_filename("local-file.txt") # List blobs for blob in bucket.list_blobs(prefix="data/"): print(blob.name, blob.size) # BigQuery operations from google.cloud import bigquery bq_client = bigquery.Client() query = """ SELECT name, COUNT(*) as cnt FROM `bigquery-public-data.usa_names.usa_1910_2013` GROUP BY name ORDER BY cnt DESC LIMIT 10 """ results = bq_client.query(query).result() for row in results: print(row.name, row.cnt) # Pub/Sub operations from google.cloud import pubsub_v1 publisher = pubsub_v1.PublisherClient() topic_path = publisher.topic_path("my-project", "my-topic") future = publisher.publish(topic_path, data=b"Hello GCP") subscriber = pubsub_v1.SubscriberClient() subscription_path = subscriber.subscription_path("my-project", "my-sub") def callback(message): print(f"Received: {message.data}") message.ack() streaming_pull = subscriber.subscribe(subscription_path, callback=callback)
